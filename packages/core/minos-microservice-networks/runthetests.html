Run the testsÔÉÅ

In order to run the tests, please make sure you have the Docker Engine and Docker Compose installed.

Move into tests/ directory

cd tests/

Run service dependencies:

docker-compose up -d

Install library dependencies:

make install

Run tests:

make test
